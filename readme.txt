JustFootball is a single page application, which provides data about several football teams.
The information includes:
	-	List of players, their nationalitiy, position, jersey number, end date of contract with the team, and market value
	-	League placement
	-	Upcomming games

The application uses data, provided by http://www.football-data.org/index, processed by using AJAX, then sent to an engine to be processed and sent to a visualizer.
HTML, HTML Grid, CSS, JQuery were used for the visualization.
